> 基于 Vue 2 + UniApp 开发的生物三节律（体力、情绪、智力）计算与可视化小程序，支持同时输入男生与女生出生日期，周期曲线及排卵日展示。


---

## 📖 项目简介

生物三节律理论认为，人的体力、情绪、智力分别以 **23天、28天、33天** 为周期波动。本小程序帮助用户：

- 🏃 **体力节律**：了解身体能量周期，规划运动与休息
- 😊 **情绪节律**：掌握情绪波动规律，调节心理状态  
- 🧠 **智力节律**：把握思维活跃期，安排学习创作

---

## ✨ 功能特性

- 📅 **日期计算**：同时输入男生与女生出生日期自动计算三节律状态
- 📈 **曲线可视化**：ECharts 绘制自定义天数周期曲线图，并支持滑动展示
- 📊 **排卵日可视化**：根据月经周期计算排卵日期，给备孕的朋友提供参考
- 💾 **历史记录**：缓存上一次输入参数，后续查询可直接使用

---

## 🚀 快速开始

### 环境要求

- HBuilderX 3.0+
- 微信开发者工具
- Node.js 12+

### 安装运行

```bash
# 克隆项目
git clone https://github.com/lovekey111/tools-uniapp.git

# 进入目录
cd tools-uniapp

# 安装依赖
npm install

# HBuilderX 运行
# 1. 打开 HBuilderX
# 2. 文件 → 打开目录 → 选择项目
# 3. 运行 → 运行到小程序模拟器 → 微信开发者工具
